## Development

使用 `pnpm` 脚本管理常用操作，命令详情与项目架构见 [README.md](./README.md)：

| 命令                     | 动作                                  |
| :----------------------- | :------------------------------------ |
| `pnpm install`           | 安装依赖                              |
| `pnpm dev`               | 启动本地开发服务器 `localhost:4321`  |
| `pnpm build`             | 构建生产站点到 `./dist/`             |
| `pnpm preview`           | 本地预览构建产物                      |
| `pnpm exec astro check`  | 类型检查                              |
